# AI 繪圖風格提示詞庫 (AI Drawing Style Prompt Library)

這是一個專為 AI 繪圖設計的提示詞庫工具，提供多種預設風格，幫助使用者快速生成高品質且具備特定美感的 AI 圖像。

## 🌟 功能特色

- **豐富的風格選擇**：收錄 32 種風格，涵蓋商業顧問 (McKinsey)、企業插畫 (Corporate Memphis)、動漫、像素藝術、復古海報、賽博龐克等多樣化領域。
- **現代化介面**：採用 Material Design 設計準則，介面簡潔且具備質感。
- **動態色彩主題**：使用者可以自由切換主題色，體驗 Dynamic Color 帶來的視覺變化。
- **一鍵複製**：點擊按鈕即可快速複製提示詞，立即投入 AI 繪圖工具（如 Midjourney, DALL-E, Stable Diffusion）使用。
- **詳細文件**：每個風格皆配備詳細的 Markdown 說明文件（位於 `style_config/`），解釋核心特色與適用場景。

## 🚀 快速開始

1. 下載或複製本專案。
2. 直接在瀏覽器中開啟 `index.html`。
3. 選擇您喜愛的風格，點擊「複製提示詞」。
4. 將提示詞貼入您的 AI 繪圖工具中。

## 📂 專案結構

- `index.html`: 主網頁應用程式，包含 UI 介面與風格資料。
- `style_config/`: 包含 32 個風格的詳細說明文件。
- `AGENTS.md`: 專案開發規範與 AI 代理指令。

## 🎨 風格類別

- **商業/專業**: 顧問風、企業風、極簡風、專業簡報風。
- **藝術/插畫**: 水彩風、等距風、可愛風、剪紙風。
- **遊戲/影視**: 動漫風、像素風、黏土風、塗鴉風。
- **復古/經典**: 達文西風、黑板風、底片風、普普藝術。
- **技術/特殊**: 藍圖風、地鐵圖風、賽博龐克、玻璃擬態風。
