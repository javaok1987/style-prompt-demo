# Style 30 ✨ 平面攝影風 Flat Lay Photography

## 核心特色

- **俯視排列視角**：採用由上往下的鳥瞰視角，將所有物件平鋪於平面上呈現。
- **真實物件呈現**：以真實生活中的物件為主角，具備高度的真實感與細節。
- **整齊有序擺放**：物件經過精心排列，呈現出網格化或對稱的秩序美感。
- **DIY 生活感**：營造出一種居家、手作且具備個人風格的生活氛圍。

## Prompt 範例

```text
Create a Flat Lay Photography style infographic about "Home Organization Tips".

Style specifications:
- Top-down bird's eye view
- Arranged real objects
- Clean light background
- DIY aesthetic
- Organized and visually pleasing layout

Aspect ratio: 9:16 vertical
```
