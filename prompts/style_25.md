# Style 25 🔬 線框圖風 Wireframe UI

## 核心特色

- **介面線框結構**：以 UI 線框稿形式呈現，強調介面配置與元件關聯。
- **灰階配色**：採用專業的灰階色彩系統，排除顏色干擾，專注於結構與流程。
- **流程邏輯清晰**：透過箭頭與佈局清楚呈現數位產品的操作邏輯。
- **數位產品感**：具備現代軟體開發的藍圖美感，展現專業的產品開發氛圍。

## Prompt 範例

```text
Create a UI Wireframe style infographic about "App Design Process".

Style specifications:
- App blueprint wireframe aesthetic
- Gray-scale color scheme
- Logic flow visualization
- Web/mobile design style
- Clean structured layout

Aspect ratio: 9:16 vertical
```
